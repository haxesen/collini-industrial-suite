# Projekt Állapot és Változásnapló

## Aktuális fázis: **Béta / Polishing**
A projekt alapvető funkciói stabilak, a vizuális megjelenés a végleges ipari designt tükrözi.

## Frissítési Napló (Legutóbbiak)

### 2026.05.12
- **Logbook Helyreállítás**: A sérült táblázat elrendezés és stílusjegyek teljes körű javítása.
- **START/KÉSZ Gombok**: Gyors gombok hozzáadása a munkafolyamat követéséhez.
- **Design Fix**: A kalkulátor visszatért a klasszikus "Yellow Card" designhoz, megőrizve a modern ticker-t.
- **Dokumentáció**: Létrejött a `docs/` mappa a technikai és stílusbeli útmutatókkal.
- **Backup Rendszer**: PowerShell script implementálása a biztonsági mentésekhez.

### 2026.05.11
- **Supabase Integráció**: Felhő alapú adatkezelés a termékekhez és naplóhoz.
- **Multilingual**: HU/DE nyelvválasztó és perzisztens tárolás.
- **PWA**: Webalkalmazás telepíthetősége mobil eszközökre.
